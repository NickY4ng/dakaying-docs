数据工坊 UI 设计文档

V0.2 | 2026-05-11

一、分析模板页

1.1 正常状态

  • 左侧导航「分析模板」高亮
  • 主区域标题「请选择模板，填写条件，开启深度分析」
  • 系统模板区：卡片网格布局，紫色定位图标 + 模板名称

1.2 无权限置灰示例（仅 1-2 个）

┌─────────────────────────────┐
│  [灰色定位图标]  上下游分析   │
│                             │
│  暂无权限，请联系管理员开通   │
│  ─────────────────────────  │
│  如需使用此模板，请确认已采购 │
│  相关数据权限               │
└─────────────────────────────┘

样式规则：

  • 卡片整体透明度 40%
  • 图标变为灰色（#999999）
  • 模板名称颜色变为 #999999
  • 小字提示：字号 12px，颜色 #FF6B6B（红色），位于卡片底部
  • 鼠标悬停：指针变为 not-allowed，无点击响应

文案：

  • 主提示：「暂无权限，请联系管理员开通」
  • 副提示：「如需使用此模板，请确认已采购相关数据权限」

二、数据下载页

2.1 页面结构

  • 左侧导航「数据下载」高亮
  • 主区域分为：条件填写区（上方）+ 下载预览区（下方）

2.2 静态提示（弹窗内）

┌─────────────────────────────────────────────┐
│  ℹ️  提示：单次下载最大支持 1GB 文件，      │
│      超出限制请缩小时间范围或筛选条件        │
└─────────────────────────────────────────────┘

样式规则：背景色 #FFF3CD，边框 1px solid #FFE69C，文字颜色 #856404，圆角 4px

2.3 动态判断（提交后）

  • 预估 ≤ 1GB：正常进入下载流程
  • 预估 > 1GB：弹窗提示错误，阻断下载
